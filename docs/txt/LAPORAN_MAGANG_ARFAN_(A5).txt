BAB IV ANALISA DAN PEMBAHASAN 4.1 Konsep Teori 4.1.1 ReactJS ReactJS adalah front-end JavaScript library yang efisien dan fleksibel untuk pengembangan antaramuka pengguna. Reactjs dikelola oleh Facebook sebagai JavaScript library sumber terbuka yang dapat digunakan untuk membuat views yang dijalankan di atas HTML. Reactjs tidak seperti kerangka kerja utuh yang mengelola seluruh aspek dari pola seper Mode-View-Controller (MVC), tetapi hanya JavaScript library sehingga hanya menangani penyusunan dan eksekusi view dari halaman web. 4.1.2 JavaScript JavaScript adalah bahasa pemrograman web yang bersifat Client Side Programming Language. Client Side Programming Language adalah tipe bahasa pemrograman yang pemrosesannya dilakukan oleh client. Aplikasi client yang dimaksud merujuk kepada web browser seperti Google Chrome, Mozilla Firefox, Opera Mini dan sebagainya. JavaScript pertama kali dikembangkan pada pertengahan tahun 90an. Meskipun memiliki nama yang hampir serupa, JavaScript berbeda dengan bahasa pemrograman Java. Untuk penulisannya, JavaScript dapat disisipkan di dalam dokumen HTML ataupun dijadikan dokumen tersendiri yang kemudian diasosiasikan dengan dokumen lain yang dituju. JavaScript mengimplementasikan fitur yang dirancang untuk mengendalikan sebuah halaman web berinteraksi dengan penggunanya. 4.1.3 CSV File Comma Separated Values (CSV) adalah file teks biasa yang berisi daftar data. File ini terkadang bisa disebut Character Separated Values atau Comma Delimited atau Comma Delimited files. File-file ini sering digunakan untuk bertukar data antara aplikasi yang berbeda. Pada umumnya file CSV menggunakan karakter koma untuk memisahkan antar data, tetapi terkadang menggunakan karakter titik koma. File CSV dapat memindahkan data dari satu sistem ke sistem lainnya dengan lebih mudah, dan tanpa melakukan input manual satu-persatu.
